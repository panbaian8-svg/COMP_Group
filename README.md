## Task 1 - OOP Application Development

A cafeteria ordering system demonstrating Object-Oriented Programming concepts:
- **Abstraction** - Abstract base class `AbstractDish`
- **Encapsulation** - Private attributes with controlled access
- **Inheritance** - `MainDish` and `Drink` subclasses
- **Polymorphism** - Method overriding for category-specific pricing

##  Task 2 - Self-Study Report
New Data Structure: Graph
Undirected graph using adjacency list for dish pairing recommendations.
New Algorithm: Radix Sort
LSD Radix Sort for sorting price values with O(d×n) time complexity.
